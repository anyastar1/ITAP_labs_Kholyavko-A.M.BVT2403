Контрольные вопросы:
1.	Java является компилируемым или интерпретируемым языком?
-	Java является одновременно и компилируемым, и интерпретируемым языком программирования. Компилируется c помощью javac  в байткод, а затем байткод выполняется виртуальной машиной JVM. JVM интерпретирует байт-код и преобразует его в машинные команды конкретной операционной системы и процессора с помощью JIT-компилятора.
2.	Что такое JVM и для чего предназначается?
-	Java Virtual Machine – виртуальная машина, абстрактная вычислительная машина, которая обеспечивает среду выполнения для Java байт кода. Она также обеспечивает запускаемость кода на любом устройстве, отвечает за освобождение памяти, занятой неиспользуемыми объектами при работе программы, а еще среда JVM безопасна для системы, так как код выполняется только в пределах этой среды.
3.	Каков жизненный цикл программы на языке Java?
-	Жизненный цикл программы на Java включает в себя сле-
дующие этапы.
1. Написание кода в текстовом редакторе или интегрированной среде разработки (IDE) с использованием синтаксиса Java.
2. Компиляция: исходный код (.java) компилируется в байт-код (.class) с помощью компилятора javac. Байт-код является платформенно независимым и может выполняться на любой платформе, где установлена JVM.
3. Интерпретация: байт-код выполняется виртуальной машиной Java. JVM интерпретирует байт-код в машинный код во время выполнения. Затем программа выходит из цикла.
4.	Какие виды типов данных есть в языке Java?
-	В Java  существуют два типа данных примитивные и ссылочные.
Примитивные делятся в свою очередь на логические - boolean и числовые - целочисленные (byte, short, int, long), вещественные (float, double), символьный - char. Ссылочные типы представляют собой объекты и массивы. Они хранят ссылки на места в памяти, где находятся данные. К основным ссылочным типам относятся классы, интерфейсы, массивы, перечисления.
5.	Чем примитивные типы данных отличаются от ссылочных?
- примитивные хранят непосредственно значение, они занимают фиксированное количество байтов (например, int занимает 4 байта). Примитивные переменные хранятся на стеке, что обеспечивает быструю доступность и освобождение памяти после завершения работы. Ссылочные же хранят ссылку (адрес в памяти) на объект, за счет чего медленнее, а также размер передаваемой ссылки фиксирован, но размер самого объекта может быть любым.
6.	Как происходит преобразование примитивных типов в Java?
- В зависимости от направления приведения различают: явное и неявное преобразование типов. При неявном приведении меньший тип данных преобразуется в больший без потери информации. Например:
• byte → short;
• short → int;
• int → long;
• float → double.
1. int a = 10;
2. long b = a; // Автоматически преобразуем int в long
Когда нужно преобразовать больший тип данных в меньший,необходимо использовать явное приведение, так как может произойти потеря точности. При этом программист берет на себя ответственность за возможные ошибки:
int → short;
long → int;
double → float.
1. long c = 1000000L;
2. int d = (int)c; // Явное приведение long к int

4. double e = 123.456;
5. float f = (float)e; // Приведение double к float
7.	Что такое байт-код в Java, и почему он важен для платформенной независимости?
-	Байт-код - это промежуточный, машинно-независимый код, представляющий собой набор инструкций для JVM. Это результат компиляции исходного Java-кода. Байт-код одинаков независимо от того, на какой операционной системе или процессоре был скомпилирован исходный код. Вся ответственность за адаптацию к конкретной платформе лежит на JVM. Достаточно написать и скомпилировать программу один раз, а затем она может быть выполнена на любой платформе, для которой существует реализация JVM. JVM выступает в роли абстрактного слоя между байт-кодом и реальным "железом".
8.	Какой тип данных используется для хранения символов в Java? Как представляются символы в памяти?
-	Для хранения одиночных символов используется примитивный тип char. Символы в Java представлены в кодировке UTF-16 (Unicode Transformation Format). Это означает, что каждый символ типа char занимает в памяти 2 байта (16 бит). Это позволяет хранить символы не только английского алфавита, но и практически всех мировых языков.
9.	Что такое литералы в Java? Приведите примеры литералов для разных типов данных.
-	Литералы представляют собой  фиксированные значения, которые непосредственно представлены в исходном коде без каких-либо вычислений.
Логические литералы: true,false;
Целочисленные :100L
Вещественные : 0.800е45
10.	 Почему Java считается строго типизированным языком? 
- потому что тип каждой переменной и выражения известен на этапе компиляции. Компилятор строго проверяет соответствие типов и не позволяет выполнять операции между несовместимыми типами без явного преобразования.
11.	 Какие проблемы могут возникнуть при использования неявного преобразования типов?
- Потеря точности при работе с целыми и дробными числами, переполнение, неочевидное поведение в составных выражениях.
Пример:
1. float k = 123456789f;
2. int l = (int)k;
3. System.out.println(l);//Выведет123456792 (потеряточности)
